# lib_proj
# Lib Project — сайт библиотеки (Django)

Веб-приложение библиотеки/каталога книг на Django: пользователи могут регистрироваться и входить в систему, просматривать каталог книг, открывать страницу конкретной книги и оформлять заказ.

---

## Возможности

- Регистрация и авторизация пользователей
- Просмотр каталога книг
- Страница книги с подробной информацией
- Оформление заказа на книгу (через форму)
- (Опционально) отзывы/оценки и форма обратной связи
- Адаптивный интерфейс (Bootstrap)

---

## Технологии

- **Backend:** Python, Django
- **Database:** SQLite (по умолчанию)
- **Frontend:** HTML, CSS, Bootstrap
- **Интерактивность:** JavaScript (валидация/лайки-дизлайки — если реализовано)

---

## Структура страниц

1. **Главная** — приветствие, навигация, быстрый доступ к каталогу  
2. **Каталог** — список книг (карточки/таблица)  
3. **Страница книги** — описание + форма заказа  
4. **Обратная связь** — форма для сообщений (если включено)
